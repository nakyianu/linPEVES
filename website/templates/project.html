Linux Privilege Escalation and Vulnerability Exploit Script (linPEVES)

linPEVES is a privilege escalation script inspired by linPEAS, a well known linux privilege escaltion vulnerability scanner.

What is Privilege Escalation?

Like any human design, computers have weaknesses which are known as vulnerabilities. Privilege Escalation refers to a class of vulnerabilities that allow an unprivileged user to gain privileges that grant them more access or control over a system. One example of this might be an account with root access who stores their account password in a file that is accessible to unprivileged users. These unprivileged users can exploit this vulnerability and escalate privileges by signing into the root userâ€™s account.

Our Project

Our project identifies and exploits these vulnerabilities to escalate a user's privileges within a Linux system. The objective of our software is to allow a user to inquire if a vulnerability exists within a system so that the possibility of privilege escalation can be determined and subsequently executed (or not).

  • Download and Install linPEVES

  • Change directory to linPEVES/
